Introduction

A utility-first CSS framework packed with classes like flex, pt-4, justify-center and bg-cyan-500 that can be composed to build any design, directly in your markup.
Get started Even with the same color palette and sizing scale, it's easy to build the same component with a completely different look in the next project.

Getting Started

You can directly import the css file in your html using the following link tag. 
            
              @import url("https://.css");
              
                <link rel="stylesheet" 
                href="https://.css">
