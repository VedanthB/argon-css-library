Images

Images can be responsive to fit the parent's width, and also can be customized to be round shaped. You can add the class img-responsive to make your image fit the width of container. It's height will get adjusted by keeping the aspect ratio same. If you want to change the aspect ratio, you will have to crop the image. You can add class img-round to make your image round shaped.
